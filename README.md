# Fast and furious | Server

## Description

This is the server for Fast and furious.

## Document

http://localhost:3000/api-doc/#/

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
